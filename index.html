Elementos de Páginas Web

Orígenes de la Web

1. Web 2.0

La Web 2.0 se refiere a una evolución de la web donde los usuarios pueden interactuar y colaborar, crear contenido y compartir información. Ejemplos incluyen redes sociales y blogs.

2. Web 3.0

La Web 3.0 es una visión de la próxima generación de la web, centrada en la inteligencia artificial y la descentralización, donde los datos están interconectados de manera más eficiente.

3. La W3C y los estándares

La W3C (World Wide Web Consortium) es una organización que desarrolla estándares para asegurar que la web sea accesible, interoperable y evolutiva.

Lenguajes de Etiquetas

1. SGML

El SGML (Standard Generalized Markup Language) es un estándar de marcado que sirvió como base para otros lenguajes de marcado como HTML y XML.

2. XML

XML (eXtensible Markup Language) es un lenguaje que define un conjunto de reglas para la codificación de documentos en un formato legible por humanos y máquinas.

3. HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Estructura el contenido y define su presentación.

4. XHTML

XHTML (eXtensible HyperText Markup Language) es una versión más estricta y bien formada de HTML, que combina las reglas de XML con HTML.

5. HTML5

HTML5 es la última versión de HTML, que incluye nuevas características como elementos multimedia, gráficos y APIs para aplicaciones web.

Otros Lenguajes Relacionados al Desarrollo Web

1. CSS/CSS3

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación de un documento HTML. CSS3 es la última versión que incluye nuevas características.

2. JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

3. Ajax

Ajax (Asynchronous JavaScript and XML) es una técnica que permite la actualización asíncrona de páginas web, mejorando la experiencia del usuario.

4. Tecnologías y lenguajes del lado del servidor

Incluyen lenguajes como PHP, Python y Ruby, que procesan las solicitudes del cliente en el servidor.

5. Navegadores

Los navegadores web, como Chrome, Firefox y Safari, interpretan el HTML, CSS y JavaScript para mostrar páginas web.

6. Motores renderizados

Los motores de renderizado son componentes de los navegadores que interpretan el código HTML y CSS para generar la visualización de las páginas.

Herramientas de Desarrollo

1. Software WYSIWYG

Herramientas como Adobe Dreamweaver permiten a los usuarios diseñar páginas web visualmente sin necesidad de escribir código.

2. Otras alternativas

  • Figma
  • InVision
  • Zeplin
  • GitHub

3. Inspección de código desde el navegador

Los navegadores ofrecen herramientas de desarrollo que permiten inspeccionar y modificar el HTML y CSS de una página en tiempo real.

El Diseño De Una Página Web

I. Tipos de páginas Web

Existen varios tipos de páginas web, incluyendo sitios informativos, portafolios, blogs, tiendas en línea y redes sociales.

II. Puntos básicos en la elaboración de una página Web

Al diseñar una página web, es fundamental considerar la usabilidad, la accesibilidad, la responsividad y el SEO.

Estructura De Una Página Web

I. Jerarquía de directorio de un proyecto web

La jerarquía de un proyecto web generalmente incluye carpetas para HTML, CSS, JavaScript, imágenes y otros recursos.

II. Elementos de estructura para el diseño de una página Web

La estructura básica de un archivo web incluye el doctype, el elemento html, el head (con metadatos) y el body (contenido visible).
